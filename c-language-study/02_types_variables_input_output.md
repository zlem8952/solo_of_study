# 2일차 - 자료형, 변수, 상수, 입력과 출력

## 오늘의 목표

- C의 기본 자료형을 이해한다.
- 변수를 선언하고 초기화할 수 있다.
- 상수와 리터럴의 차이를 안다.
- `scanf`, `printf`의 기본 사용법을 익힌다.
- 형 변환과 오버플로의 감각을 익힌다.

## 변수란?

변수는 `데이터를 저장하는 메모리 공간에 붙인 이름`입니다.

```c
int age = 20;
double height = 175.3;
char grade = 'A';
```

위 코드는 다음 의미입니다.

- `age`는 정수형 변수
- `height`는 실수형 변수
- `grade`는 문자형 변수

## 기본 자료형

| 자료형 | 의미 | 예시 |
| --- | --- | --- |
| `char` | 문자 또는 작은 정수 | `'A'`, `65` |
| `int` | 정수 | `10`, `-3` |
| `short` | 더 작은 정수 | `100` |
| `long` | 더 큰 범위의 정수 | `100000L` |
| `long long` | 더 큰 정수 | `123456789LL` |
| `float` | 단정도 실수 | `3.14f` |
| `double` | 배정도 실수 | `3.141592` |
| `long double` | 더 큰 정밀도의 실수 | `3.14L` |

주의:

- 정확한 바이트 수는 컴파일러와 시스템에 따라 다를 수 있습니다.
- 하지만 일반적으로 `char <= short <= int <= long <= long long` 관계를 가집니다.

## 부호 있는 정수와 부호 없는 정수

```c
signed int a = -10;
unsigned int b = 10;
```

- `signed`는 음수와 양수를 표현합니다.
- `unsigned`는 음수가 없고 0 이상의 값만 표현합니다.

주의:

```c
unsigned int x = 1;
printf("%u\n", x - 2);
```

이 코드는 음수가 아니라 매우 큰 양수처럼 보일 수 있습니다. 부호 없는 정수는 언더플로 시 기대와 다르게 보일 수 있습니다.

## 변수 선언과 초기화

```c
int a;
a = 10;

int b = 20;
int c = b + 5;
```

초기화하지 않은 지역 변수는 쓰레기 값을 가질 수 있습니다.

```c
int main(void)
{
    int x;
    printf("%d\n", x); // 위험: 초기화되지 않음
    return 0;
}
```

## 상수와 리터럴

### 리터럴

코드에 직접 적은 값입니다.

```c
10
3.14
'A'
"Hello"
```

### `const`

값이 바뀌지 않아야 하는 변수에 사용합니다.

```c
const double PI = 3.1415926535;
```

```c
PI = 4.0; // 오류
```

### 매크로 상수

```c
#define MAX_SIZE 100
```

이 방식은 전처리기가 문자 치환을 하는 방식입니다. 차이는 10일차에서 자세히 다룹니다.

## 정수 리터럴 표기

```c
int a = 10;    // 10진수
int b = 012;   // 8진수
int c = 0x12;  // 16진수
```

주의:

- `012`는 십이(12)가 아니라 8진수입니다.
- 초보자가 자주 헷갈립니다.

## 문자와 아스키 코드

```c
char ch = 'A';
printf("%c\n", ch);
printf("%d\n", ch);
```

- `%c`로 출력하면 문자 `A`
- `%d`로 출력하면 문자 코드 값이 나옵니다.

## 문자열

문자열은 문자 배열로 저장됩니다.

```c
char name[] = "Alice";
```

실제로는 끝에 널 문자 `'\0'`가 자동으로 붙습니다.

메모리 모습:

```text
'A' 'l' 'i' 'c' 'e' '\0'
```

## `sizeof` 연산자

어떤 자료형이나 객체의 크기를 바이트 단위로 구합니다.

```c
printf("%zu\n", sizeof(int));
printf("%zu\n", sizeof(double));
```

`sizeof`의 결과 타입은 `size_t`이므로 보통 `%zu`를 사용합니다.

## 입력과 출력

### `printf`

```c
int age = 25;
double weight = 70.5;

printf("age = %d\n", age);
printf("weight = %.1f\n", weight);
```

### `scanf`

```c
int age;
scanf("%d", &age);
```

중요:

- `scanf`는 변수의 값을 바꾸기 위해 `주소`를 받아야 하는 경우가 많습니다.
- 그래서 `&age`처럼 `&`를 붙입니다.

### 여러 값 입력

```c
int a, b;
scanf("%d %d", &a, &b);
```

### 실수 입력

```c
double x;
scanf("%lf", &x);
```

주의:

- `float`는 `scanf`에서 `%f`
- `double`은 `scanf`에서 `%lf`
- `printf`에서는 `float`도 `double`처럼 처리되므로 `%f`를 사용합니다.

## 예제: 원의 넓이 계산

```c
#include <stdio.h>

int main(void)
{
    double radius;
    const double PI = 3.1415926535;

    printf("반지름을 입력하세요: ");
    scanf("%lf", &radius);

    printf("원의 넓이 = %.2f\n", PI * radius * radius);
    return 0;
}
```

## 형 변환

### 자동 형 변환

```c
int a = 10;
double b = 3.5;
double result = a + b;
```

여기서는 `a`가 자동으로 `double`처럼 처리됩니다.

### 명시적 형 변환

```c
double x = 3.9;
int y = (int)x;
```

`y`는 `3`이 됩니다. 소수 부분은 잘립니다.

## 정수 나눗셈의 함정

```c
int a = 7;
int b = 2;
printf("%d\n", a / b); // 3
```

정수끼리 나누면 결과도 정수입니다.

실수 결과를 원하면:

```c
printf("%.2f\n", (double)a / b); // 3.50
```

## 오버플로와 언더플로

정수 범위를 넘는 계산은 위험합니다.

```c
int x = 2147483647;
x = x + 1; // 시스템에 따라 문제가 생길 수 있음
```

부호 있는 정수 오버플로는 정의되지 않은 동작이 될 수 있습니다. 즉, 믿을 수 없는 결과가 나옵니다.

## 자주 쓰는 서식 문자열 정리

| 용도 | `printf` | `scanf` |
| --- | --- | --- |
| `int` | `%d` | `%d` |
| `unsigned int` | `%u` | `%u` |
| `char` | `%c` | `%c` |
| 문자열 | `%s` | `%s` |
| `float` | `%f` | `%f` |
| `double` | `%f` | `%lf` |
| 16진수 | `%x` | `%x` |
| 주소 | `%p` | 없음 |

## 예제: 학생 정보 입력

```c
#include <stdio.h>

int main(void)
{
    int age;
    double score;
    char grade;

    printf("나이 점수 학점을 입력하세요: ");
    scanf("%d %lf %c", &age, &score, &grade);

    printf("나이: %d\n", age);
    printf("점수: %.2f\n", score);
    printf("학점: %c\n", grade);

    return 0;
}
```

주의:

- `%c`는 공백 문자도 읽습니다.
- 줄바꿈이 남아 있으면 예상과 다르게 동작할 수 있습니다.

위 예제에서 `%lf`와 `%c` 사이의 공백은 의도적으로 넣은 것입니다. `scanf`의 공백 문자는 입력에 남아 있는 줄바꿈이나 스페이스를 건너뛰게 만듭니다.

문자 하나만 따로 읽을 때는 보통 아래처럼 씁니다.

```c
scanf(" %c", &grade);
```

반대로 공백을 넣지 않으면 직전에 남아 있던 줄바꿈을 바로 읽어 버릴 수 있습니다.

```c
scanf("%c", &grade);
```

문자 입력 처리는 5일차, 11일차에서 더 자세히 다룹니다.

## 실수 포인트

- `scanf`에서 `&`를 빼먹음
- `double` 입력에 `%f`를 씀
- 정수 나눗셈인데 실수 결과를 기대함
- 초기화하지 않은 변수를 사용함
- `char`와 문자열을 혼동함

## 오늘 정리

- 자료형은 메모리 해석 방법입니다.
- 변수는 메모리 공간에 이름을 붙인 것입니다.
- 입력 함수는 보통 값을 바꾸기 위해 주소를 받습니다.
- 형 변환과 정수 나눗셈은 초보자가 가장 많이 실수하는 부분입니다.
